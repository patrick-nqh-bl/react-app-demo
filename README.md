# Patrick's project (using MERN Stack)
1. How to install
-------
* Github CLI
-------
#### Clone source code 
> git clone https://github.com/patrick-nqh-bl/react-app-demo.git
#### Install packages
> git install
#### Start database server
> cd backend 
> nodemon server
#### Start react server
> react-app-demo-1> npm start
App running....
